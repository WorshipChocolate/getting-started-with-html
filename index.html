Important Definitions

HTML(HyperText Mark-Up Language): the main document type of the web
URL: how you refer to documents on the web
HTTP: the protocol that unites the web; the web runs on this
World Wide Web: a collection of HTML documents; files can be almost anything
Browsers: Things like Internet Explorer, Google Chrome, or Firefox
Servers: Computers that host instead of doing the browsing. They host most of the files that make up the Web.
Internet: the world's largest computer network
Tag: a command contained within angled brackets. There are usually opening and closing tags. Some may not require closing. These are called Void Tags.
Elements: everything within a set of opening and closing tags.
Attribute: a property of a HTML element

HTML

HTML, or HyperText Mark-Up Language, is the main document type of web. It is made of text content (what you see), mark-up what it looks like), references to other documents, and links to other pages. The HTML mark up is made of elements, which are made of tags.

Tags

In tags, there is an opening tag and an closing sign. Each is surrounding by greater/less than signs, which within dictate which command you are using. In between these tags are the words you wish to write. The only difference between an opening and closing tag is the slash mark before the command. (Ex: /Name)

There are many different types of tags. Here are the basic ones and their commands:

B-Tag (Bold Tag), represented by a b.
Em-Tag (Em. stands for emphasis; Italics), represented by em.

HTML Attributes

Represented by tag ATTR= "value"; ending is /tag. The ATTR stands for attribute.

A-Tag (Anchor Tag, links), represented by a href= "url" the closing tag is /a.
Img-Tag (Image Tag, images), represented by img src= "url" alt= "text" This is a void tag. Void tags need no closing tags. The src stands for source, and alt stands for alternate. This part is "required"- it isn't really needed but you need to use it. It is used for broken requests, and is also used when blind people use the computer.

Whitespace

When typing in the elements, text ends up strung on one big line unless you use one of two specific commands.

text on ----------------> text on multiple lines

multiple lines

Br-Tag (Break Tag, a line break; Void Tag), represented by br. Remember, Void Tags need no closing tag.

P-Tag (Paragraph Tag, a block) represented by p. When you use this tag, a box seems to appear around the word selected, creating a slight line break where it is used. The length and width of the (invisible) tag can be changed to fit your needs.

Inline VS Block

HTML documents are composed of two types of elements- inline or block. Block elements form an invisible box around the content.

Inlines include: b, em, img, br, etc.
